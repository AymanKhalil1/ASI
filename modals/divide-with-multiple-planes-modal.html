  3. Divide with Multiple Planes

The Divide option, available in the Modify tab, now allows dividing objects into multiple ones using multiple planes. The dividing planes must be parallel to the XY, XZ, or YZ planes. The resulting objects' meshes are automatically redefined.

Using the Divide option will launch the Divide dialog box. Using the + symbol next to the plane field will add a new plane to the list. The planes can be removed by clicking the - symbol next to the plane field.

Using the text fields, the user can define the position of the dividing planes. The position is defined using the perpendicular distance (shift) from the global planes.

The divide command is executed upon clicking the OK button, and the selected object is divided using the specified planes
