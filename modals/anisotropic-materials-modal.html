Open this page in a new tab

ELS V11 adds the ability to model using custom defined anisotropic materials, i.e., materials with different properties in different directions. This is useful for modeling materials like wood, fiber-reinforced composites, quartz, as well as some arrangements of reinforced concrete that can exhibit different physical properties such as crack formation behavior, thermal expansion, and elastic moduli in different directions.

The new option is available in the Preprocessing Browser. The user can create custom groups of elements and assign different materials to them depending on the axis of interest.

For maximum customization, users can define their own materials in the Material Generator and use those for the anisotropic material's axes.
