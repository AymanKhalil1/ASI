  • Create Region on Segments
    ELS V11 introduces a new utility that allows users to create 8-Node regions on segments of a hybrid polyline. This is specially useful when working with multiplanar complex geometries and requiring a precise definition of a region's boundaries. The user can then use in any preprocessing action, e.g. changing spring materials along a certain path, or removing them entirely and thus inducing a crack.

    To use this utility, select any hybrid polyline draft, click the Additional Tasks button, choose Convert Draft To , then Create 8-Node Regions on Segments .

  • Dowel Action
    New properties added in the Properties Browser for any structural component that contains reinforcement and/or stirrups. If enabled, the new properties, an rft dowel action toggle and a stirrup dowel action toggle, replace the single spring that normally represents an RFT rebar or stirrup branch with four springs to allow for nonuniform stress distribution in a single rebar's cross‚Äêsection (i.e. rebars will allow for some bending moment resistance).

  • Object Local Axes

    New option to in the ELS Viewer to toggle the visibility of local axes of individual objects added to the Properties Browser.

    Addtionally, the Contours dialog now has a new option of calculating contours in the direction of local axes.

    The ELS Viewer also allows the user to cycle the directions of the local axes of any wall object.

  • Spring-related Utility Updates
    • Replace Spring Material by Location
      New option added in Preprocessing to change the spring material for selected element groups. The user can specify the location of the springs intended for the material changing action: between elements of the same object, between elements of different objects, or both.

      In the following example, the elements (purple outline) assigned to the group are the same in all three cases, but the set of springs that changes materials (purple colored) differs in each case based on the chosen location.

      In the first case shown below, the springs that get changed when the Choose Springs Across Objects are chosen are the ones that link two different objects, such as girder to slab (green highlight) or column to girder (yellow highlight).

      In the second case, the springs that get changed when the Choose Springs in Same Object option is chosen are the ones that link elements of the same objects, such as the ones between elements of the slab (yellow highlight), the ones between elements of the column (blue highlight), or the ones between elements of the girder (green highlight).

      In the third case, the springs that get changed when the All option is chosen are all of the springs that link elements of the same object or two different objects, for example, the ones belonging to a single object (between slab elements ‚Äê blue highlight), and the ones joining different objects (between girder and slab elements ‚Äê yellow highlight).

    • Spring Controller Filters
      New Option added in Spring Controller groups to designate spring relative displacement reaching a certain magnitude as a trigger for the spring altering actions performed through the Spring Controller .

  • Style Collection Properties
    The new Style Collection Properties dialog allows users to manage the properties of style collections in ELS. This includes editing the name, description, and style components of every individual descendant style without having to navigate to its Properties dialog box.

  • Sync Forces and Ground Reactions
    New option added in Visualization tab to show an indicator for synchronous forces imported as loading from the analysis of another ELS model, as well as an indicator for ground reactions. The new indicators' scale can be adjusted from the Drawing Size Settings dialog box.
