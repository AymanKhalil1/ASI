  3. Append Elements

All preprocessing groups now support the ability to append elements to previously selected and assigned elements.

The user does not need to multi-select (Ctrl + Click/Drag selection area) the new group of elements. The append feature automatically reselects a group's elements in the background, so the user can focus on selecting elements to append.
