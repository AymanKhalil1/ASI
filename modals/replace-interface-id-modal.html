Replace Interface ID

ELS V11 allows the user to switch the interface ID of a group of elements, and subsequently, the material of the springs that bind them to the adjacent elements. The new feature can be reached by double-clicking the Replace Interface ID option in the Preprocessing Browser or choosing the same option from the ribbon's Analysis tab.

    To replace the interface ID of a group of elements, follow these steps:
  1. Click New and Apply to create a new group.
  2. Select the newly created group in the left-hand side tree to navigate to the group properties window.
  3. In the group properties window, click the Select button to return to the modeling window and select the elements, then press ENTER.
  4. From the dropdown menu, choose the interface you want to assign to the group, then click OK.

As always, the user can navigate to the Interface Materials dialog (now in the Materials tab) to modify the material of each interface combination.
