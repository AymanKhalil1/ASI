Import Materials

The new Material Generator in ELS V11 allows the user to select the exact materials to import in their project. This allows the user to select the exact ones they need, and leave out the ones they don't need or already have in the project (e.g. existing default materials) to avoid duplication. To import materials into your project, follow these steps:

  1. Open the Import Materials dialog by navigating to the Material Types Browser, right-clicking on the topmost item (Material Generator) in the material list, and choosing Import.
  2. After clicking Import, a File Explorer window opens up where you can navigate to the location of the .mat file exported from another project and open it.
  3. In the Import Materials dialog, select the materials to import using their corresponding checkboxes.

Import Styles

In the same manner, ELS V11 allows the user to select the exact styles to import into their project. To import styles into your project, follow these steps:

  1. Open the Import Styles dialog by navigating to the Style Types Browser, right-clicking on the topmost item (Style Generator) in the style list, and choosing Import.
  2. After clicking Import, a File Explorer window opens up where you can navigate to the location of the .stl file exported from another project and open it.
  3. In the Import Styles dialog, select the styles to import using their corresponding checkboxes.
