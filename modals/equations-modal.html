Equations

The Equations feature in ELS parlance refers to the ability to define and manipulate mathematical formulae which contain multiple physical quantities (e.g., stress, strain, displacement, etc.) and then use these formulae to compute new quantities. These new quantities can then be visualized in the ELS Viewer or used in other analyses.

Modularized Equations Feature

ELS V11 introduces a modularized approach to equations, allowing for greater flexibility and customization in result analysis, and broader use with more ELS Viewer components. The new Equations feature is available from the Tools tab in the ELS Viewer.

Equations can be created for:

  1. Elements
  2. Springs
  3. Straining Actions

Equation Generator Dialog

Clicking one of the Equation options opens the Equation Generator dialog, where you can create, edit, and manage equations.

The Equation Generator dialog allows you to define equations using a simple syntax, which can include mathematical operations, functions, and physical quantities. To create a new equation, click the New button.

In order to start building the equation, rename the equation in the Equation Name field in the table, then press the Details button.

  1. Click the upper Add button to add variables.
  2. Choose a physical quantity to assign to each new variable from the Type dropdown.
  3. Use the provided calculator-like interface to build a mathematical expression using the created variables, which correspond to physical quantities.
  4. To add variables to the expression field, select a variable from the Variable Type dropdown.
  5. Press OK to add the new equation, then OK again to close the Equation Generator dialog.
