    The Ribbon

    The introduction of the ribbon interface in ELS V11 marks a significant milestone in the ELS user experience. By replacing traditional menus with an intuitive ribbon layout, users can now access tools and features more efficiently. The ribbon organizes commands in a logical, visual manner, making it easier for both new and experienced users to find and utilize the functionality they need. This modern approach to interface design reduces the learning curve while maintaining quick access to all essential modeling tools. The ribbon's contextual tabs and grouped commands provide a clearer overview of available features, eliminating the need to navigate through nested menu structures. This streamlined approach helps users focus more on their modeling tasks and less on searching for commands.

    The diagram below shows a high level overview of the Ribbon tabs and the some of functions that can be done through each one. The orange-colored items are some of the most common tasks, so they are extracted to the Home ribbon tab.

    The Home tab contains some of the most frequently used commands from the Draw, Modify, Edit, View, Materials, and Analysis tabs. These commands are often used multiple times, at more than one project phase, so they have been conveniently placed in the Home Tab.

    The Draw Tab lists all the building pieces that can be used to model a structure. These include Draft Objects that can be used to help with modeling structural components or labeling a structure, as well as Structural Objects that represent physical structural components.

    The Draw tab commands are:

    • New Level
      Inserts a new blank level into the Structure Browser, and opens the Level Properties dialog box for the created level.
    • Properties
      Opens the Structure Properties dialog box.
    • Point
      Draws a single point which can be used as a reference, or a start or end position specifier for other objects.
