    User Interface

    The ELS interface is designed to ensure the best possible work efficiency while youâ€™re modeling your project, running the analysis or even documenting the results through professional reports. The interface is intuitive and user-friendly, allowing you to focus on your modeling tasks without unnecessary distractions. This section will focus on overviewing the main components of the Modeler, while the Viewer's Interface will be explained in detail in the Viewer Manual. The main components of the ELS Modeler interface are:

    • The Ribbon
    • The Quick Access Toolbar
    • The Browser Panel
    • The Modeling window

    The Ribbon

    The introduction of the ribbon interface in ELS V11 marks a significant milestone in the ELS user experience. By replacing traditional menus with an intuitive ribbon layout, users can now access tools and features more efficiently. The ribbon organizes commands in a logical, visual manner, making it easier for both new and experienced users to find and utilize the functionality they need. This modern approach to interface design reduces the learning curve while maintaining quick access to all essential modeling tools. The ribbon's contextual tabs and grouped commands provide a clearer overview of available features, eliminating the need to navigate through nested menu structures. This streamlined approach helps users focus more on their modeling tasks and less on searching for commands.

    The diagram below shows a high level overview of the Ribbon tabs and the some of functions that can be done through each one. The orange-colored items are some of the most common tasks, so they are extracted to the Home ribbon tab.

    The Home tab contains some of the most frequently used commands from the Draw, Modify, Edit, View, Materials, and Analysis tabs. These commands are often used multiple times, at more than one project phase, so they have been conveniently placed in the Home Tab.

    The Draw Tab lists all the building pieces that can be used to model a structure. These include Draft Objects that can be used to help with modeling structural components or labeling a structure, as well as Structural Objects that represent physical structural components.

    The Modify tab commands are:

    • Copy
    • Copy Format Settings
      enables the user to modify the properties of the target object to match the properties of another object according to the ticked properties in the copy format dialog (Style, material, RFT extension,...etc.).
    • Mirror
    • Shift
    • Offset
    • Scale
    • Scale 3D
    • Trim
    • Extend
    • Group Trim/Extend
    • Explode
    • Array
    • Extrusion
    • Revolution
    • Align Edges
    • Divide
    • Intersect
      a tool to remove the overlap between structural objects by specifying Cutter objects and Objects to be cut.
    • Crack
      opens cracks in structural objects with a user specified depth. The cracks are defined using drafts and the object with the crack has to be a structural object.
    • Create Hole
      opens holes in structural objects with a user specified depth. The holes are defined using closed drafts.
    • Chamfer
    • Fillet
    • Follow Me
      converts a 2D draft into a 3D element (8-node) by extruding it along a path [Unlike Extrusion, the path can be completely as per the user preference. It can be a straight line, a series of straight lines, a curve, a series of curves or a combination of them all]. (For drafts only)
    • Create Hinge
    • Create Nails at Contact Surface
      allows the user to quickly generate nails at interfaces between chosen groups of structural components. The generated nails are generated as drafts which can be later converted to bolts or nails of any chosen material and dimensions.
    • Create Region at Contact Surface
      creates an 8-node region at the contact surface between two structural objects.
    • Align
      Adjusts the position of the selected objects relative to one another based on one of the following 9 criteria: minimum x, maximum x, center x, minimum y, maximum y, center y, minimum z, maximum z, center z.
    • Extract Elements
      Separates the elements from their original structural object and converts them into 8-node elements. If the meshing of the object is an automatic mesh, it will be converted to a custom mesh.
    • Merge Element Lists
      a tool to combine(merge) a number of elements of a structural object in one element in cases where mesh-adjustment is needed.
    • Re-mesh Selected Elements
      a tool to re-divide an already existing mesh whether through intensifying the mesh density at specific locations or changing the mesh type.
    • Shift Element Corners
      enables the user to push or pull elements or objects corners a specific distance.
    • Append to Component
      a tool to merge the elements of a structural object to be part of another structural object taking the type of the "appended to" object but with a custom mesh.

    The Tools tab commands are:

    • Check Integrity
      a tool to detect overlaps between structure objects that might affect the solution outputs and remove those overlaps by specifying cutter objects and the objects to be Cut.
    • Mesh Calculator
      displays all mesh related info for all structural objects in the model.
    • Blocks
      a tool to group several objects together to act as if one object. Blocks allow object groups to be duplicated within the same project or across multiple projects by importing/exporting them
    • Min/Max Elements Volume
      adjusts the element-volume maximum and minimum bounds.
    • Motion Path
      a tool to record AVI videos to present the different parts and angles of the structure on ELS.
    • Attach Files
      enables the user to make attachments to an ordinary ELS file. This command is useful when the user needs to link project-related papers or presentations to the current ELS file.
    • Objects Attachment
      used to attach two objects together.
    • Replace Assigned Material/Style
    • Purge Unused Assignments
      removes unused assignments to eliminate the unneeded clutter.
    • Filtration
      aids the user to filter the objects based on a set of filtration criteria.
    • Reports
      enables the user to create detailed report(s) of their current project on ELS presenting the model itself as snapshots, materials, RFT, load assignments,lumped masses,....etc. The reports generated are in pdf format and can be totally customized as per the user's preferences.
    • Units
      specifies the units used in the model and correspondingly the results.
    • Light Settings
      adjusts the displayed light settings in the modeling window along with setting the virtual location of the light source.
    • Command Line Settings
      displays the commands shortcuts and enables the user to edit or create new shortcuts to run certain commands which rapid up the modelling process.
    • Preferences
      enables the user to adjust the main features affecting his/her experience on dealing with of ELS's interface including General, Wall, General Steel Section, RFT, Slab, Implicit Steel Plate, Link, 8-Node, Window, Girder, Column & Circular Column preferences.

    The Analysis tab commands are:

    • Analysis Settings
      enables the user to edit the general settings for maximum output customization. It also allows the user to enables the user to edit the restrains of the degrees of freedom and the gravity options that are to be applied to the whole structure.
    • Loading Scenario
      enables the user to add/remove loading stages and edit them.
    • Output Files Options
      Specifies the types of data calculated by the solver and enabled for viewing in the Viewer.
    • Contact Options
    • Lumped Mass
      displays and creates lumped mass out of the selected elements or region(s).
    • Lumped Damping
      specifies the damping factor (coefficient) of the selected elements.
    • Rigid Body
      displays and creates rigid bodies out of adjacent elements or out of a region.
    • Element Springs
      displays and adjusts the number multiplied to the normal number of springs on each face.
    • Replace Element Material
      grants the user the ability to customize element materials. The enhanced Elements Material Properties editor allows the user to control the choice of material for different element groups.
    • Replace Spring Material
      grants the user the ability to customize a spring group's material.
    • Anisotropic Materials
      allows the user to change the material for a group of elements in each local direction. This is useful for anisotropic materials such as wood, which may have different properties in different directions.
    • Construct Scenario
      allows the user to designate elements for construction during any point in the simulation timeline.
    • Demolish Scenario
      allows the user to designate elements for demolition during any point in the simulation timeline.
    • Replace Interface ID
      allows the user to switch the assigned interface ID for a group of elements at analysis time.
    • Spring Controller
      enables the user to view the spring types in the current model and adjust their action (activate, change material, remove......etc.).
    • Vehicle Manager
      used to assign dynamic vehicle loads to the model.
    • Elements Data Query
      enables the user to view the data assigned to the selected element.
    • Physical Data Conflict
    • Rigid Body Conflict
      highlights groups containing common elements that are assigned to rigid bodies as well as boundary conditions, elastic boundary conditions, initial velocity, initial acceleration, static displacement load, or dynamic displacement load. ELS would not typically allow for such a conflict to exist if elements are chosen directly. However, if elements are chosen using 8-Node regions, these conflicts can arise because the assignment of rigid bodies to elements within a region is deferred to the analysis stage.
    • Run Solver
      runs the analysis (solves the structural analysis problem).
    • Resume Backup Solver
      allows the user to display the backups specified through the results viewer of the current project.
    • Stop Solver
      allows the user to stop the solver before the analysis ends.
    • Iterative Solver Settings
      allows the user to control and edit the iterative solver settings.
    • Open ELS Viewer
    • Explore Output Folder
    • Delete Output
    • Separation Groups
      allows the user to view the groups separated in ELS's results viewer after running the analysis.

    Browsers

    Project Description

    The Project Description tab enables the user to edit the project's general properties.

    Mesh Calculator

    The Mesh Calculator dialog aims at displaying the mesh-related info.

    Material Quantities

    The Material Quantities form displays each object's material volumetric quantities along with the total material quantities used in the project.

    Level Details

    The Level Details tab enables the user to activate/deactivate layers, add, delete and set levels active.

    Style Generator

    The Style Generator dialog aims at displaying the available styles in each category (Columns, girders, etc.).

    Style Components

    The Style Components dialog aims at displaying each style component (Geometry, RFT & Steel Section) in detail for each structural component.
